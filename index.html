My Project Idea

For my project idea I would like to create a virtual reality application that is able to imitate many real-life activities. This application will also imitate real life physics and other things as such. I think this project is interesting because I find the idea of virtual reality fascinating as it allows users to experience the virtual world as though they are living in it. The main objective of this application will be to create a simulation of workplace tasks that can be monitored by employers or trainers. This can range from a variety of labour-intensive jobs such as engineering to tradie jobs. This project will be interesting because virtual reality is an experience everyone enjoys including me. Creating a virtual world that imitates the looks and physics of the real-world will be interesting as I would be able to compare the two different worlds side by side. In relation to imitating the real world, this can be a great way to assess employees or potential employees to see whether they are correctly and safely performing their work without the risk of accident. In 2017 the number of work-related fatalities added up to a total of 190 according to SafeWorkAustralia. With the help of virtual reality training/assessments it can reduce the number of employees unfit for their jobs from risking injury and it can allow them to learn of possible accidents that may occur in the workplace. The features of this service include realistic physics, graphics, noises and minimal implementation of realistic feeling/touch. This will be achieved though the use of Unreal engine which will provide a close to real life looking 3d world and 3d models. The physics or this will also closely replicate those of real life. Advantages of these are that users are able to immerse themselves into the virtual world as though they were actually at the workplace so that. Furthermore props that replicate the feel of real tools such as drills, hammers, screwdrivers, etc will have sensors programmed to simulate movement in the virtual world. The user will be able to see and feel these tools as though they were working, keeping in mind that the machinery such as saws and drills, and heavy machinery such as forklifts and excavators will not have working motors to ensure the safety of the user. Another feature of this will be motion capture gloves with sensors that capture enough data to closely replicate the movement of the userâ€™s hands. The movements of the user will also be captured by motion sensors on their feet which will allow them to travel around the virtual world without sacrificing hand space which will be used for performing tasks, unlike other virtual reality games which use handheld controllers. The sounds will be recorded from the same workplaces as the virtual world to again give an immersive feel. With the machinery that may be used in the workplace, replicas of the cabs will be made with working controls that are the same as real life. The software used, as said previously will be Unreal engine and various Autodesk applications, and the hardware that will be used will include an oculus rift, which is the virtual reality headset, motion capture gloves which can be source from multiple companies such as CyberGlove systems, custom made replica tools and machinery to replicate real ones with motion capture sensors, a powerful computer able to run vr at moderate graphics and a motion capture studio. The skills required to successfully achieve my project are heavily reliant on programming which is done in all aspects of the software used. Experience in the hardware used must be evident as much of it is very complex, especially the motion capture technology. The hardware may be difficult to source because it is extremely expensive to fund as well as the creation of custom pieces, which follows onto the question of the possibility of executing this project. I do not believe I will be able to achieve because of cost and accessibility, as well as the realism of this project however I do find this very interesting and hope it does become possible. However an alternative way to achieve this project with less cost and effort would be to use vr controllers instead of motion tracking. This will be a far more feasible option as it is easier to access and program. If this project is successful it can be used to train labour intensive employees as well as test their skills in the workplace without risking serious injury. This project could save lives if it is successful and furthermore can also branch off to create almost fully immersive games.
